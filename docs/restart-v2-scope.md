# TVBox Restart v2 首发范围决策

## 版本定位

首发版本定位为：**电视端 Leanback 稳定测试版**。

目标不是一次性做完整商业系统，而是先验证四件事：

1. App 能稳定安装和启动
2. 数据源能配置和播放
3. 数据源到期后有清晰、温和的更新引导
4. 用户能通过公众号拿到新数据源并重新配置

## 首发包含

- 电视端 Leanback arm64-v8a
- 数据源配置时间记录
- 15 天有效期判断
- 到期/即将到期提示
- 公众号更新引导
- 复制公众号名称
- 打开微信入口
- 基础安装与更新说明文档

## 首发暂缓

- 手机端 Mobile 包
- Python 爬虫恢复
- DLNA 投屏替代方案
- 会员系统
- 登录系统
- 复杂远程后台

## 体验原则

- 不使用强压迫式文案
- 不强调“锁死”，改成“配置已过期，请获取最新数据源”
- 保留刷新状态入口
- 公众号文案保持简单明确

## 技术原则

- 保持 `main` 分支始终可编译
- `restart-v2` 分支先做可测试增强
- 每次功能修改后至少跑一次 Leanback arm64-v8a Debug 编译
- Release 签名前不把 Debug 包当正式发布包
